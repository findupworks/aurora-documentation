# Configurando Canais de Comunicação

Os canais definem onde e como seus agentes estarão disponíveis para interação.

## Tipos de Canais

### Chat Web

Interface de chat integrada na plataforma.

### Chat Embed

Widget para incorporar em sites e aplicações.

### Email

Respostas automáticas por email.

### Webhooks

Integração com sistemas externos via HTTP.

## Configurando um Canal

1. Abra o agente desejado
2. Vá até a aba **Channels**
3. Selecione o canal que deseja configurar
4. Preencha as configurações específicas
5. Salve as alterações

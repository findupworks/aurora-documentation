[Confluence](https://www.atlassian.com/software/confluence) é a poderosa plataforma de colaboração em equipe e gestão de conhecimento da Atlassian. Serve como um espaço de trabalho centralizado onde as equipes podem criar, organizar e compartilhar informações entre departamentos e organizações.

Com o Confluence, você pode:

- **Criar documentação estruturada**: Construir wikis completos, planos de projetos e bases de conhecimento com formatação rica
- **Colaborar em tempo real**: Trabalhar em conjunto em documentos com colegas de equipe, com comentários, menções e capacidades de edição
- **Organizar informações hierarquicamente**: Estruturar o conteúdo com espaços, páginas e hierarquias aninhadas para navegação intuitiva
- **Integrar com outras ferramentas**: Conectar com Jira, Trello e outros produtos da Atlassian para integração perfeita do fluxo de trabalho
- **Controlar permissões de acesso**: Gerenciar quem pode visualizar, editar ou comentar em conteúdos específicos

No Sim, a integração com o Confluence permite que seus agentes acessem e aproveitem a base de conhecimento da sua organização. Os agentes podem recuperar informações das páginas do Confluence, buscar conteúdo específico e até mesmo atualizar a documentação quando necessário. Isso permite que seus fluxos de trabalho incorporem o conhecimento coletivo armazenado na sua instância do Confluence, tornando possível criar agentes que possam consultar documentação interna, seguir procedimentos estabelecidos e manter recursos de informação atualizados como parte de suas operações.

## Instruções de uso

Integre o Confluence no fluxo de trabalho. Pode ler e atualizar uma página. Requer OAuth.

## Ferramentas

### `confluence_retrieve`

Recupera conteúdo das páginas do Confluence utilizando a API do Confluence.

#### Entrada

| Parâmetro | Tipo   | Obrigatório | Descrição                                                                                             |
| --------- | ------ | ----------- | ----------------------------------------------------------------------------------------------------- |
| `domain`  | string | Sim         | Seu domínio do Confluence \(ex.: suaempresa.atlassian.net\)                                          |
| `pageId`  | string | Sim         | ID da página do Confluence a recuperar                                                                |
| `cloudId` | string | Não         | ID do Confluence Cloud para a instância. Se não fornecido, será obtido utilizando o domínio.         |

#### Saída

| Parâmetro | Tipo   | Descrição                                       |
| --------- | ------ | ----------------------------------------------- |
| `ts`      | string | Marca de tempo da recuperação                   |
| `pageId`  | string | ID da página do Confluence                      |
| `content` | string | Conteúdo da página com tags HTML removidas      |
| `title`   | string | Título da página                                |

### `confluence_update`

Atualiza uma página do Confluence utilizando a API do Confluence.

#### Entrada

| Parâmetro | Tipo   | Obrigatório | Descrição                                                                                             |
| --------- | ------ | ----------- | ----------------------------------------------------------------------------------------------------- |
| `domain`  | string | Sim         | Seu domínio do Confluence \(ex.: suaempresa.atlassian.net\)                                          |
| `pageId`  | string | Sim         | ID da página do Confluence a atualizar                                                                |
| `title`   | string | Não         | Novo título para a página                                                                             |
| `content` | string | Não         | Novo conteúdo para a página em formato de armazenamento do Confluence                                 |
| `version` | number | Não         | Número de versão da página \(necessário para prevenir conflitos\)                                     |
| `cloudId` | string | Não         | ID do Confluence Cloud para a instância. Se não fornecido, será obtido utilizando o domínio.         |

#### Saída

| Parâmetro | Tipo    | Descrição                                         |
| --------- | ------- | ------------------------------------------------- |
| `ts`      | string  | Marca de tempo da atualização                     |
| `pageId`  | string  | ID da página do Confluence                        |
| `title`   | string  | Título da página atualizado                       |
| `success` | boolean | Status de sucesso da operação de atualização      |

[Notion](https://www.notion.so) é um espaço de trabalho tudo-em-um que combina notas, documentos, wikis e ferramentas de gerenciamento de projetos em uma única plataforma. Oferece um ambiente flexível e personalizável onde os usuários podem criar, organizar e colaborar em conteúdo em vários formatos.

Com o Notion, você pode:

- **Criar conteúdo versátil**: Construir documentos, wikis, bancos de dados, quadros kanban, calendários e mais
- **Organizar informações**: Estruturar conteúdo hierarquicamente com páginas aninhadas e bancos de dados poderosos
- **Colaborar sem problemas**: Compartilhar espaços de trabalho e páginas com membros da equipe para colaboração em tempo real
- **Personalizar seu espaço de trabalho**: Projetar seu fluxo de trabalho ideal com modelos flexíveis e blocos de construção
- **Conectar informações**: Vincular entre páginas e bancos de dados para criar uma rede de conhecimento
- **Acessar de qualquer lugar**: Usar o Notion em plataformas web, desktop e móveis com sincronização automática

No Sim, a integração com o Notion permite que seus agentes interajam diretamente com seu espaço de trabalho do Notion de forma programática. Isso permite cenários de automação poderosos como gerenciamento de conhecimento, criação de conteúdo e recuperação de informações. Seus agentes podem:

- **Ler páginas do Notion**: Extrair conteúdo e metadados de qualquer página do Notion.
- **Ler bancos de dados do Notion**: Recuperar a estrutura e informações de bancos de dados.
- **Escrever em páginas**: Adicionar novo conteúdo a páginas existentes do Notion.
- **Criar novas páginas**: Gerar novas páginas do Notion sob uma página principal, com títulos e conteúdo personalizados.
- **Consultar bancos de dados**: Buscar e filtrar entradas de bancos de dados usando critérios avançados de filtragem e ordenação.
- **Buscar no espaço de trabalho**: Buscar em todo seu espaço de trabalho do Notion páginas ou bancos de dados que correspondam a consultas específicas.
- **Criar novos bancos de dados**: Criar programaticamente novos bancos de dados com propriedades e estrutura personalizadas.

Esta integração fecha a lacuna entre seus fluxos de trabalho de IA e sua base de conhecimento, permitindo gerenciamento de documentação e informações sem problemas. Ao conectar o Aurora Workflow com o Notion, você pode automatizar processos de documentação, manter repositórios de informações atualizados, gerar relatórios e organizar informações de forma inteligente, tudo através de seus agentes inteligentes.

## Instruções de uso

Integre o Notion no fluxo de trabalho. Pode ler páginas, ler bancos de dados, criar páginas, criar bancos de dados, adicionar conteúdo, consultar bancos de dados e buscar no espaço de trabalho. Requer OAuth.

## Ferramentas

### `notion_read`

Ler conteúdo de uma página do Notion

#### Entrada

| Parâmetro | Tipo   | Obrigatório | Descrição                         |
| --------- | ------ | ----------- | --------------------------------- |
| `pageId`  | string | Sim         | O ID da página do Notion para ler |

#### Saída

| Parâmetro  | Tipo   | Descrição                                                                                     |
| ---------- | ------ | --------------------------------------------------------------------------------------------- |
| `content`  | string | Conteúdo da página em formato markdown com cabeçalhos, parágrafos, listas e tarefas pendentes |
| `metadata` | object | Metadados da página incluindo título, URL e marcas de tempo                                   |

### `notion_read_database`

Ler informações e estrutura de banco de dados do Notion

#### Entrada

| Parâmetro    | Tipo   | Obrigatório | Descrição                                 |
| ------------ | ------ | ----------- | ----------------------------------------- |
| `databaseId` | string | Sim         | O ID do banco de dados do Notion para ler |

#### Saída

| Parâmetro  | Tipo   | Descrição                                                                                        |
| ---------- | ------ | ------------------------------------------------------------------------------------------------ |
| `content`  | string | Informações do banco de dados incluindo título, esquema de propriedades e metadados              |
| `metadata` | object | Metadados do banco de dados incluindo título, ID, URL, marcas de tempo e esquema de propriedades |

### `notion_write`

Adicionar conteúdo a uma página do Notion

#### Entrada

| Parâmetro | Tipo   | Obrigatório | Descrição                                          |
| --------- | ------ | ----------- | -------------------------------------------------- |
| `pageId`  | string | Sim         | O ID da página do Notion à qual adicionar conteúdo |
| `content` | string | Sim         | O conteúdo para adicionar à página                 |

#### Saída

| Parâmetro | Tipo   | Descrição                                                               |
| --------- | ------ | ----------------------------------------------------------------------- |
| `content` | string | Mensagem de sucesso que confirma que o conteúdo foi adicionado à página |

### `notion_create_page`

Criar uma nova página no Notion

#### Entrada

| Parâmetro  | Tipo   | Obrigatório | Descrição                                            |
| ---------- | ------ | ----------- | ---------------------------------------------------- |
| `parentId` | string | Sim         | ID da página pai                                     |
| `title`    | string | Não         | Título da nova página                                |
| `content`  | string | Não         | Conteúdo opcional para adicionar à página ao criá-la |

#### Saída

| Parâmetro  | Tipo   | Descrição                                                                 |
| ---------- | ------ | ------------------------------------------------------------------------- |
| `content`  | string | Mensagem de sucesso que confirma a criação da página                      |
| `metadata` | object | Metadados da página incluindo título, ID da página, URL e marcas de tempo |

### `notion_query_database`

Consultar e filtrar entradas de banco de dados do Notion com filtragem avançada

#### Entrada

| Parâmetro    | Tipo   | Obrigatório | Descrição                                                  |
| ------------ | ------ | ----------- | ---------------------------------------------------------- |
| `databaseId` | string | Sim         | O ID do banco de dados a consultar                         |
| `filter`     | string | Não         | Condições de filtro em formato JSON (opcional)             |
| `sorts`      | string | Não         | Critérios de ordenação como array JSON (opcional)          |
| `pageSize`   | number | Não         | Número de resultados a retornar (padrão: 100, máximo: 100) |

#### Saída

| Parâmetro  | Tipo   | Descrição                                                                                                                  |
| ---------- | ------ | -------------------------------------------------------------------------------------------------------------------------- |
| `content`  | string | Lista formatada de entradas do banco de dados com suas propriedades                                                        |
| `metadata` | object | Metadados da consulta incluindo contagem total de resultados, informações de paginação e array de resultados sem processar |

### `notion_search`

Buscar em todas as páginas e bancos de dados no espaço de trabalho do Notion

#### Entrada

| Parâmetro    | Tipo   | Obrigatório | Descrição                                                                      |
| ------------ | ------ | ----------- | ------------------------------------------------------------------------------ |
| `query`      | string | Não         | Termos de busca (deixar vazio para obter todas as páginas)                     |
| `filterType` | string | Não         | Filtrar por tipo de objeto: página, banco de dados, ou deixar vazio para todos |
| `pageSize`   | number | Não         | Número de resultados a retornar (padrão: 100, máximo: 100)                     |

#### Saída

| Parâmetro  | Tipo   | Descrição                                                                                                                |
| ---------- | ------ | ------------------------------------------------------------------------------------------------------------------------ |
| `content`  | string | Lista formatada de resultados da busca incluindo páginas e bancos de dados                                               |
| `metadata` | object | Metadados da busca incluindo contagem total de resultados, informações de paginação e matriz de resultados sem processar |

### `notion_create_database`

Criar um novo banco de dados no Notion com propriedades personalizadas

#### Entrada

| Parâmetro    | Tipo   | Obrigatório | Descrição                                                                                                         |
| ------------ | ------ | ----------- | ----------------------------------------------------------------------------------------------------------------- |
| `parentId`   | string | Sim         | ID da página principal onde o banco de dados será criado                                                          |
| `title`      | string | Sim         | Título para o novo banco de dados                                                                                 |
| `properties` | string | Não         | Propriedades do banco de dados como objeto JSON (opcional, criará uma propriedade "Nome" padrão se estiver vazio) |

#### Saída

| Parâmetro  | Tipo   | Descrição                                                                                         |
| ---------- | ------ | ------------------------------------------------------------------------------------------------- |
| `content`  | string | Mensagem de sucesso com detalhes do banco de dados e lista de propriedades                        |
| `metadata` | object | Metadados do banco de dados incluindo ID, título, URL, tempo de criação e esquema de propriedades |

[WhatsApp](https://www.whatsapp.com/) é uma plataforma de mensagens globalmente popular que permite comunicação segura e confiável entre indivíduos e empresas.

A API do WhatsApp Business fornece às organizações capacidades poderosas para:

- **Interagir com clientes**: Enviar mensagens personalizadas, notificações e atualizações diretamente para o aplicativo de mensagens preferido dos clientes
- **Automatizar conversas**: Criar chatbots interativos e sistemas de resposta automatizada para consultas comuns
- **Melhorar o suporte**: Fornecer atendimento ao cliente em tempo real através de uma interface familiar com suporte para conteúdo multimídia
- **Impulsionar conversões**: Facilitar transações e acompanhamentos com clientes em um ambiente seguro e compatível

No Sim, a integração com o WhatsApp permite que seus agentes aproveitem essas capacidades de mensagens como parte dos seus fluxos de trabalho. Isso cria oportunidades para cenários sofisticados de interação com clientes como lembretes de compromissos, códigos de verificação, alertas e conversas interativas. A integração conecta seus fluxos de trabalho de IA com os canais de comunicação dos clientes, permitindo que seus agentes entreguem informações oportunas e relevantes diretamente aos dispositivos móveis dos usuários. Ao conectar o Aurora Workflow com o WhatsApp, você pode construir agentes inteligentes que interajam com os clientes através de sua plataforma de mensagens preferida, melhorando a experiência do usuário enquanto automatiza tarefas rotineiras de mensagens.

## Instruções de uso

Integre o WhatsApp no fluxo de trabalho. Pode enviar mensagens.

## Ferramentas

### `whatsapp_send_message`

Enviar mensagens do WhatsApp

#### Entrada

| Parâmetro       | Tipo   | Obrigatório | Descrição                                             |
| --------------- | ------ | ----------- | ----------------------------------------------------- |
| `phoneNumber`   | string | Sim         | Número de telefone do destinatário com código do país |
| `message`       | string | Sim         | Conteúdo da mensagem a enviar                         |
| `phoneNumberId` | string | Sim         | ID do número de telefone do WhatsApp Business         |

#### Saída

| Parâmetro     | Tipo    | Descrição                                          |
| ------------- | ------- | -------------------------------------------------- |
| `success`     | boolean | Status de sucesso do envio da mensagem do WhatsApp |
| `messageId`   | string  | Identificador único da mensagem do WhatsApp        |
| `phoneNumber` | string  | Número de telefone do destinatário                 |
| `status`      | string  | Status de entrega da mensagem                      |
| `timestamp`   | string  | Marca de tempo do envio da mensagem                |

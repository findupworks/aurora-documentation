[Perplexity AI](https://www.perplexity.ai) é um motor de busca e respostas impulsionado por IA que combina as capacidades de grandes modelos de linguagem com buscas web em tempo real para fornecer informações precisas e atualizadas, bem como respostas completas a perguntas complexas.

Com o Perplexity AI, você pode:

- **Obter respostas precisas**: Receber respostas completas a perguntas com citações de fontes confiáveis
- **Acessar informações em tempo real**: Obter informações atualizadas através das capacidades de busca web do Perplexity
- **Explorar tópicos em profundidade**: Aprofundar-se em tópicos com perguntas de acompanhamento e informações relacionadas
- **Verificar informações**: Verificar a credibilidade das respostas através das fontes e referências fornecidas
- **Gerar conteúdo**: Criar resumos, análises e conteúdo criativo baseado em informações atuais
- **Pesquisar eficientemente**: Agilizar processos de pesquisa com respostas completas a consultas complexas
- **Interagir conversacionalmente**: Participar de diálogos naturais para refinar perguntas e explorar tópicos

No Sim, a integração do Perplexity permite que seus agentes aproveitem essas capacidades poderosas de IA de forma programática como parte dos seus fluxos de trabalho. Isso permite cenários de automação sofisticados que combinam compreensão de linguagem natural, recuperação de informações em tempo real e geração de conteúdo. Seus agentes podem formular consultas, receber respostas completas com citações e incorporar essas informações em seus processos de tomada de decisão ou resultados. Esta integração fecha a lacuna entre a automação do seu fluxo de trabalho e o acesso a informações atuais e confiáveis, permitindo que seus agentes tomem decisões mais informadas e forneçam respostas mais precisas. Ao conectar o Aurora Workflow com o Perplexity, você pode criar agentes que se mantenham atualizados com as informações mais recentes, forneçam respostas bem pesquisadas e entreguem insights mais valiosos aos usuários - tudo sem exigir pesquisa manual ou coleta de informações.

## Instruções de uso

Integre o Perplexity no fluxo de trabalho. Pode gerar completions usando os modelos de chat do Perplexity AI. Requer chave API.

## Ferramentas

### `perplexity_chat`

Gera completions usando os modelos de chat do Perplexity AI

#### Entrada

| Parâmetro      | Tipo   | Obrigatório | Descrição                                                       |
| -------------- | ------ | ----------- | --------------------------------------------------------------- |
| `systemPrompt` | string | Não         | Instrução do sistema para guiar o comportamento do modelo       |
| `content`      | string | Sim         | O conteúdo da mensagem do usuário para enviar ao modelo         |
| `model`        | string | Sim         | Modelo a usar para os completions de chat (ex.: sonar, mistral) |
| `max_tokens`   | number | Não         | Número máximo de tokens a gerar                                 |
| `temperature`  | number | Não         | Temperatura de amostragem entre 0 e 1                           |
| `apiKey`       | string | Sim         | Chave API do Perplexity                                         |

#### Saída

| Parâmetro | Tipo    | Descrição                        |
| --------- | ------- | -------------------------------- |
| `success` | boolean | Status de sucesso da operação    |
| `output`  | object  | Resultados do completion de chat |

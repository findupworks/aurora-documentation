[OpenAI](https://www.openai.com) é uma empresa líder em pesquisa e implantação de IA que oferece uma suíte de modelos e APIs de IA poderosos. A OpenAI fornece tecnologias de ponta que incluem modelos de linguagem grandes (como GPT-4), geração de imagens (DALL-E) e embeddings que permitem aos desenvolvedores criar aplicações sofisticadas impulsionadas por IA.

Com a OpenAI, você pode:

- **Gerar texto**: Crie texto semelhante ao humano para diversas aplicações usando modelos GPT
- **Criar imagens**: Transforme descrições textuais em conteúdo visual com DALL-E
- **Produzir embeddings**: Converta texto em vetores numéricos para busca semântica e análise
- **Construir assistentes de IA**: Desenvolva agentes conversacionais com conhecimento especializado
- **Processar e analisar dados**: Extraia insights e padrões de texto não estruturado
- **Traduzir idiomas**: Converta conteúdo entre diferentes idiomas com alta precisão
- **Resumir conteúdo**: Condense texto extenso preservando informações-chave

No Sim, a integração da OpenAI permite que seus agentes aproveitem essas capacidades poderosas de IA de forma programática como parte dos seus fluxos de trabalho. Isso permite cenários de automação sofisticados que combinam compreensão de linguagem natural, geração de conteúdo e análise semântica. Seus agentes podem gerar embeddings vetoriais a partir de texto, que são representações numéricas que capturam o significado semântico, permitindo sistemas avançados de busca, classificação e recomendação. Além disso, através da integração com DALL-E, os agentes podem criar imagens a partir de descrições textuais, abrindo possibilidades para geração de conteúdo visual. Esta integração fecha a lacuna entre sua automação de fluxos de trabalho e as capacidades de IA de ponta, permitindo que seus agentes entendam o contexto, gerem conteúdo relevante e tomem decisões inteligentes baseadas em compreensão semântica. ## Instruções de uso

Integre Embeddings no fluxo de trabalho. Pode gerar embeddings a partir de texto. Requer chave API.

## Ferramentas

### `openai_embeddings`

Gerar embeddings a partir de texto usando OpenAI

#### Entrada

| Parâmetro        | Tipo   | Obrigatório | Descrição                                       |
| ---------------- | ------ | ----------- | ----------------------------------------------- |
| `input`          | string | Sim         | Texto para gerar embeddings                     |
| `model`          | string | Não         | Modelo a utilizar para os embeddings            |
| `encodingFormat` | string | Não         | O formato em que os embeddings serão retornados |
| `apiKey`         | string | Sim         | Chave API da OpenAI                             |

#### Saída

| Parâmetro | Tipo    | Descrição                           |
| --------- | ------- | ----------------------------------- |
| `success` | boolean | Status de sucesso da operação       |
| `output`  | object  | Resultados da geração de embeddings |

Com o Slack, você pode:

- **Automatizar notificações de agentes**: Enviar atualizações em tempo real dos seus agentes Aurora Workflow para qualquer canal do Slack
- **Criar endpoints de webhook**: Configurar bots do Slack como webhooks para ativar fluxos de trabalho do Aurora Workflow a partir de atividades do Slack
- **Melhorar fluxos de trabalho de agentes**: Integrar mensagens do Slack nos seus agentes para entregar resultados, alertas e atualizações de status
- **Criar e compartilhar canvas do Slack**: Gerar programaticamente documentos colaborativos (canvas) em canais do Slack
- **Ler mensagens de canais**: Recuperar e processar mensagens recentes de qualquer canal do Slack para monitoramento ou ativação de fluxos de trabalho

No Sim, a integração com o Slack permite que seus agentes interajam programaticamente com o Slack de várias formas como parte dos seus fluxos de trabalho:

- **Enviar mensagens**: Os agentes podem enviar mensagens formatadas para qualquer canal ou usuário do Slack, compatível com a sintaxe mrkdwn do Slack para formatação rica.
- **Criar canvas**: Os agentes podem criar e compartilhar canvas do Slack (documentos colaborativos) diretamente nos canais, permitindo compartilhar conteúdo mais rico e documentação.
- **Ler mensagens**: Os agentes podem ler mensagens recentes dos canais, permitindo monitorar, informar ou acionar ações adicionais baseadas na atividade do canal.

Isso permite cenários de automação poderosos como enviar notificações, alertas, atualizações e relatórios diretamente para o centro de comunicação da sua equipe, compartilhar documentos estruturados ou monitorar conversas para acionar fluxos de trabalho. Seus agentes podem entregar informações oportunas, compartilhar resultados de processos que completaram, criar documentos colaborativos ou alertar membros da equipe quando atenção for necessária. Esta integração fecha a lacuna entre seus fluxos de trabalho de IA e a comunicação da sua equipe, garantindo que todos se mantenham informados sem intervenção manual. Ao conectar o Aurora Workflow com o Slack, você pode criar agentes que mantenham sua equipe atualizada com informações relevantes no momento certo, melhorar a colaboração compartilhando conhecimento automaticamente e reduzir a necessidade de atualizações manuais de status, tudo enquanto aproveita seu espaço de trabalho do Slack existente onde sua equipe já se comunica.

## Instruções de uso

Integre o Slack no fluxo de trabalho. Pode enviar mensagens, criar canvas e ler mensagens. Requer OAuth. Pode ser usado no modo de ativação para iniciar um fluxo de trabalho quando uma mensagem é enviada a um canal.

## Ferramentas

### `slack_message`

Envia mensagens para canais ou usuários do Slack através da API do Slack. Compatível com o formato mrkdwn do Slack.

#### Entrada

| Parâmetro    | Tipo   | Obrigatório | Descrição                                                    |
| ------------ | ------ | ----------- | ------------------------------------------------------------ |
| `authMethod` | string | Não         | Método de autenticação: oauth ou bot_token                   |
| `botToken`   | string | Não         | Token do bot para Bot personalizado                          |
| `channel`    | string | Sim         | Canal do Slack alvo (ex.: #general)                          |
| `text`       | string | Sim         | Texto da mensagem a enviar (suporta formato mrkdwn do Slack) |
| `files`      | file[] | Não         | Arquivos para anexar à mensagem                              |

#### Saída

| Parâmetro | Tipo   | Descrição                               |
| --------- | ------ | --------------------------------------- |
| `ts`      | string | Marca de tempo da mensagem              |
| `channel` | string | ID do canal onde a mensagem foi enviada |

### `slack_canvas`

Cria e compartilha canvas do Slack em canais. Canvas são documentos colaborativos dentro do Slack.

#### Entrada

| Parâmetro          | Tipo   | Obrigatório | Descrição                                   |
| ------------------ | ------ | ----------- | ------------------------------------------- |
| `authMethod`       | string | Não         | Método de autenticação: oauth ou bot_token  |
| `botToken`         | string | Não         | Token do bot para Bot personalizado         |
| `channel`          | string | Sim         | Canal do Slack alvo (ex.: #general)         |
| `title`            | string | Sim         | Título do canvas                            |
| `content`          | string | Sim         | Conteúdo do canvas em formato markdown      |
| `document_content` | object | Não         | Conteúdo estruturado do documento do canvas |

#### Saída

| Parâmetro   | Tipo   | Descrição                      |
| ----------- | ------ | ------------------------------ |
| `canvas_id` | string | ID do canvas criado            |
| `channel`   | string | Canal onde o canvas foi criado |
| `title`     | string | Título do canvas               |

### `slack_message_reader`

Lê as últimas mensagens dos canais do Slack. Recupera o histórico de conversas com opções de filtragem.

#### Entrada

| Parâmetro    | Tipo   | Obrigatório | Descrição                                              |
| ------------ | ------ | ----------- | ------------------------------------------------------ |
| `authMethod` | string | Não         | Método de autenticação: oauth ou bot_token             |
| `botToken`   | string | Não         | Token do bot para Bot personalizado                    |
| `channel`    | string | Sim         | Canal do Slack do qual ler mensagens (ex.: #general)   |
| `limit`      | number | Não         | Número de mensagens a recuperar (padrão: 10, máx: 100) |
| `oldest`     | string | Não         | Início do intervalo de tempo (marca de tempo)          |
| `latest`     | string | Não         | Fim do intervalo de tempo (marca de tempo)             |

#### Saída

| Parâmetro  | Tipo  | Descrição                             |
| ---------- | ----- | ------------------------------------- |
| `messages` | array | Array de objetos de mensagem do canal |

[BrowserUse](https://browser-use.com/) é uma plataforma poderosa de automação de navegador que permite criar e executar tarefas de navegador de forma programática. Fornece uma maneira de automatizar interações web através de instruções em linguagem natural, permitindo que você navegue por sites, preencha formulários, extraia dados e realize sequências complexas de ações sem escrever código.

Com o BrowserUse, você pode:

- **Automatizar interações web**: Navegar para sites, clicar em botões, preencher formulários e realizar outras ações de navegador
- **Extrair dados**: Obter conteúdo de sites, incluindo texto, imagens e dados estruturados
- **Executar fluxos de trabalho complexos**: Encadear múltiplas ações para completar tarefas web sofisticadas
- **Monitorar a execução de tarefas**: Observar a execução de tarefas de navegador em tempo real com feedback visual
- **Processar resultados programaticamente**: Receber saídas estruturadas de tarefas de automação web

No Sim, a integração do BrowserUse permite que seus agentes interajam com a web como se fossem usuários humanos. Isso possibilita cenários como pesquisa, coleta de dados, envio de formulários e testes web, tudo através de simples instruções em linguagem natural. Seus agentes podem coletar informações de sites, interagir com aplicações web e realizar ações que normalmente exigiriam navegação manual, ampliando suas capacidades para incluir toda a web como recurso.

## Instruções de uso

Integre o uso do navegador no fluxo de trabalho. Pode navegar pela web e realizar ações como se um usuário real estivesse interagindo com o navegador. Requer chave API.

## Ferramentas

### `browser_use_run_task`

Executa uma tarefa de automação de navegador usando BrowserUse

#### Entrada

| Parâmetro           | Tipo    | Obrigatório | Descrição                                                           |
| ------------------- | ------- | ----------- | ------------------------------------------------------------------- |
| `task`              | string  | Sim         | O que o agente de navegador deve fazer                              |
| `variables`         | json    | Não         | Variáveis opcionais para usar como segredos (formato: (key: value)) |
| `format`            | string  | Não         | Sem descrição                                                       |
| `save_browser_data` | boolean | Não         | Se os dados do navegador devem ser salvos                           |
| `model`             | string  | Não         | Modelo LLM a utilizar (padrão: gpt-4o)                              |
| `apiKey`            | string  | Sim         | Chave API para a API do BrowserUse                                  |

#### Saída

| Parâmetro | Tipo    | Descrição                           |
| --------- | ------- | ----------------------------------- |
| `id`      | string  | Identificador de execução de tarefa |
| `success` | boolean | Status de conclusão da tarefa       |
| `output`  | json    | Dados de saída da tarefa            |
| `steps`   | json    | Passos de execução realizados       |

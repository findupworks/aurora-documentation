## Comparação rápida

| Disparador                | Condição de início                                             |
| ------------------------- | -------------------------------------------------------------- |
| **API**                   | POST HTTP autenticado                                          |
| **Chat**                  | Mensagem de implementação de chat                              |
| **Formulário de entrada** | Ao enviar manualmente no editor ou fluxo de trabalho principal |
| **Manual**                | Botão de execução no editor                                    |
| **Agendamento**           | Temporizador gerenciado no modal de agendamento                |
| **Webhook**               | Ao receber uma solicitação HTTP                                |

## Uso de disparadores

1. Coloque o bloco disparador no slot de início.
2. Configure qualquer esquema ou autenticação necessária.
3. Conecte o bloco ao resto do fluxo de trabalho.

> As implementações potencializam cada disparador. Atualize o fluxo de trabalho, reimplemente-o, e todos os pontos de entrada de disparadores capturam a nova instantânea. Saiba mais em [Execução → Instantâneas de implementação](/workflow/pt-BR/execution).

Os blocos de início herdados permanecem para fluxos existentes, mas não aparecem mais em novas compilações.

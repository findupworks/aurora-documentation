As conexões são as vias que permitem que os dados fluam entre blocos no seu fluxo de trabalho. Elas definem como a informação é passada de um bloco para outro, permitindo criar processos sofisticados de múltiplos passos.

<Callout type="info">
  Conexões corretamente configuradas são essenciais para criar fluxos de trabalho eficazes. Elas determinam como
  os dados se movem através do seu sistema e como os blocos interagem entre si.
</Callout>

# Usando Decisões e Condições

As decisões permitem criar lógica condicional em seus workflows, executando diferentes ações baseadas em condições específicas.

## Tipos de Decisões

### Decide

Avalia uma condição e retorna verdadeiro ou falso.

### Check If

Verifica se uma condição específica é verdadeira.

### Filter

Filtra dados baseado em critérios definidos.

## Criando Decisões

1. Adicione a ação "Decide" ao workflow
2. Configure a condição a ser avaliada
3. Defina ações para cada resultado (verdadeiro/falso)
4. Teste o workflow

## Exemplos

### Exemplo: Verificar Urgência

```
Se o email contém "urgente":
  → Enviar notificação imediata
Senão:
  → Adicionar à fila normal
```

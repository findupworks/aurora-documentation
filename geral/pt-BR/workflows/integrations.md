# Integrando com Sistemas Externos

Conecte seus workflows com sistemas externos através de webhooks, APIs e integrações pré-configuradas.

## Webhooks

Envie dados para sistemas externos através de requisições HTTP.

## APIs

Conecte-se a APIs REST para buscar ou enviar dados.

## Integrações Pré-configuradas

Use conectores pré-configurados para sistemas populares.

## Configurando Webhooks

1. Adicione a ação "Send webhook" ao workflow
2. Configure a URL de destino
3. Defina o método HTTP (GET, POST, etc.)
4. Configure headers e body
5. Teste a integração

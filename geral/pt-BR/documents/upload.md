# Upload e Processamento de Documentos

Aprenda como fazer upload de documentos e processá-los para uso com RAG.

## Formatos Suportados

- PDFs
- Arquivos de texto
- URLs
- Texto direto

## Fazendo Upload

1. Acesse a coleção desejada
2. Clique em **Adicionar Documento**
3. Escolha o método de upload
4. Aguarde o processamento

## Processamento

Os documentos são processados automaticamente:

- Extração de texto
- Indexação para busca
- Preparação para RAG

# Usando Starters Avançados

Os starters definem como seu flow é acionado. Aprenda a usar starters avançados para criar flows mais poderosos.

## Tipos de Starters

### Agendamento

Execute flows em horários específicos ou intervalos regulares.

### Eventos

Dispara flows quando eventos específicos ocorrem.

### Webhooks

Acione flows através de requisições HTTP externas.

## Configurando Starters

1. Ao criar um flow, escolha o starter desejado
2. Configure os parâmetros específicos
3. Teste o starter antes de ativar o flow

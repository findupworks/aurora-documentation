# Integrando Chat Embed

Incorpore o chat em seu site ou aplicação usando o widget embed.

## Obtendo o Código

1. Abra o agente desejado
2. Vá até **Channels** > **Chat Embed**
3. Gere o código de incorporação
4. Copie o código

## Instalando

1. Cole o código no seu site
2. Personalize conforme necessário
3. Teste o widget

## Personalização

- Cores e estilo
- Posição na página
- Comportamento e animações

# Configurando Chat Web

Configure o chat web integrado na plataforma.

## Ativando Chat Web

1. Abra o agente desejado
2. Vá até **Channels** > **Chat**
3. Ative o chat web
4. Configure personalizações

## Personalização

- Cores e tema
- Mensagens de boas-vindas
- Comportamento e configurações
